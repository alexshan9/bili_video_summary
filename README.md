完整运行项目，你需要有如下配置要求：
1. 一个服务器能运行本地的whisper模型。
2. 一个能调云端大模型的账号，比如 硅基流动，openrouter。推荐使用硅基流动，新用户注册送好多
3. 前端 nodejs 环境
4. 后端 python3.11

## 后端安装运行
配置设置
```bash
正确配置config.ini，根据 config.ini.examplel的参考
```
依赖安装
`pip install -r requirements.txt`
项目运行
python main.py

## 前端安装运行
依赖安装
```bash
cd webui
npm i
```

项目运行
```bash
npm run dev
```
